# Project Brief

## Multimodal Representation Learning for Scientific Data

We are exploring models that align text, graphs, and tabular scientific measurements into shared representations.
Students joining this project should be comfortable with PyTorch, experimental design, and reproducibility.

### Desired signals
- Can reproduce an existing paper
- Writes clear experiment notes
- Comfortable debugging data pipelines
- Understands ablation studies

### Screening questions
1. Tell me about a time you reproduced a paper and discovered a hidden implementation assumption.
2. What is the difference between predictive accuracy and representation usefulness?
3. How would you evaluate robustness when labels are scarce?
